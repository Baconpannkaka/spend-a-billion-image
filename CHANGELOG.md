# Changelog

## 0.4.0 — Wikimedia image workflow

- Lade till manuell GitHub Action för Wikimedia Commons-import.
- Lade till licensallowlist för CC0, public domain, CC BY och CC BY-SA.
- Stoppar NC-, ND- och okända licenser.
- Laddar ned och konverterar valda bilder till WebP.
- Sparar fotograf, källa, licens, Commons-fil, SHA-1, träffpoäng och status.
- Lade till återupptagningsbar import för 334 verifierade produkter.
- Lade till stratifierat provurval om 30 produkter.
- Lade till manuella sök- och filundantag.
- Lade till `/bildgranskning` med filtrering, alternativ och kopiering av produkt-id:n.
- Lade till separat GitHub Action för godkännande, avvisning och återställning.
- Utökade bildvalidering och tester.
- Dokumenterade hela flödet i `IMAGE-IMPORT.md`.
